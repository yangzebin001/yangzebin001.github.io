一、GIT下载、安装与配置

1.下载与安装

下载git只需要直接在官网下载就可以了，它会自动下载适合你电脑配置的版本。
官网

安装就一直下一步就可以了。

2.配置

安装完成之后右键菜单就会自动添加两个选项：

  • Git GUI
  • Git Bash

在任意目录下右键点击 Git Bash 打开Git 命令行，输入一下两条命令：

“USERNAME”填写你的用户名
“xxx@xx.com”填写你的邮箱

这样配置完后你之后所创建的仓库都会默认创建在你的名下。
